## 💡 Python Nedir?

**Python**, 1991 yılında **Guido van Rossum** tarafından geliştirilen, yüksek seviyeli, yorumlanan (interpreted) ve nesne yönelimli (OOP) bir programlama dilidir. Günümüzde dünyanın en popüler ve en çok tercih edilen dillerinden biridir.

### 🎭 İsmi Nereden Gelir?
Çoğu kişi isminin "Piton" yılanından geldiğini düşünse de, aslında Guido van Rossum bu ismi sevdiği bir İngiliz komedi grubu olan **"Monty Python's Flying Circus"**tan almıştır.

## 🚀 Neden Python?

Python'un bu kadar popüler olmasının birkaç temel sebebi vardır:

1.  **Yazım Kolaylığı (Sözdizimi):** Python, neredeyse İngilizce gibi okunabilen, karmaşık semboller yerine girintilere (indentation) dayalı temiz bir yapıya sahiptir.
2.  **Geniş Kütüphane Desteği:** Veri biliminden web geliştirmeye kadar binlerce hazır kütüphaneye (Pandas, TensorFlow, Django vb.) sahiptir.
3.  **Çok Yönlülük:** Hemen hemen her alanda kullanılabilir.
4.  **Topluluk:** Dev bir topluluğa sahip olması, karşılaştığınız her sorunun çözümünü kolayca bulmanızı sağlar.

## 🌍 Kullanım Alanları

Python günümüzde şu alanlarda başrol oynamaktadır:

-   **🤖 Yapay Zeka ve Makine Öğrenmesi:** Scikit-learn, TensorFlow, PyTorch.

-   **🌐 Web Geliştirme (Backend):** Django, Flask, FastAPI.

## 📖 Hızlı Başlangıç (Küçük Bir Tutorial)

Python'un ne kadar basit olduğunu şu örneklerle görebilirsiniz:

### 1. Ekrana Yazdırma
```python
print("Merhaba Python!")
```

### 2. Değişken Tanımlama
```python
isim = "Sadık"
yas = 25
ogrenci_mi = True
```

### 3. Liste ve Döngüler
```python
meyveler = ["Elma", "Armut", "Muz"]

for meyve in meyveler:
    print(f"Benim sevdiğim meyve: {meyve}")
```

### 4. Fonksiyon Tanımlama
```python
def selamla(ad):
    return f"Selam {ad}, hoş geldin!"

print(selamla("Sadık"))
```
